# Welcome to ResuMate

## Project info

Web-based AI-powered resume analysis platform that provides ATS-style scoring, grammar feedback, resume improvement suggestions, job role recommendations, and an interactive dashboard for tracking resume history.

**URL**: https://lovable.dev/projects/daf877b3-78ce-4d76-8493-1bc8e65b30e2


## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
